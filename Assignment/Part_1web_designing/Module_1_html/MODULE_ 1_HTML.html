(Q.5) : What are different types of lists in HTML?

Ans => In HTML, there are three main types of lists: ordered lists, unordered lists, and definition lists.

Ordered Lists:
                         

An ordered list is a numbered list, where each list item is preceded by a sequential number.
The HTML tag used to create an ordered list is <ol>. Each item in the list is enclosed in an <li> tag.

Example :
Example code for an ordered list:
                     <ol>
                            <li>First item</li>
                            <li>Second item</li>
                            <li>Third item</li>
                     </ol>
This will produce a numbered list like this:
                     1.First item
 
                     2.Second item

                     3.Third item
Unordered Lists:
                         

An unordered list is a bulleted list, where each item is preceded by a bullet point.
The HTML tag used to create an unordered list is <ul>. Each item in the list is enclosed in an <li> tag.

Example :
Example code for an ordered list:
                     <ul>
                            <li>First item</li>
                            <li>Second item</li>
                            <li>Third item</li>
                     </ul>
This will produce a numbered list like this:
                     • First item
 
                     •  Second item

                     •  Third item
Definition Lists:
                         

A definition list is a list of terms and their definitions.
The HTML tags used to create a definition list are <dl> for the entire list,
<dt> for the term, and <dd> for the definition.

Example code for an ordered list:
                     <dl>
                            <dt>Term 1</dt>
                            <dd>Definition 1</dd>
                            <dt>Term 2</dt>
                            <dd>Definition 2</dd>
                            <dt>Term 3</dt>
                            <dd>Definition 3</dd>
                   </dl>
This will produce a list like this:
                     Term 1
                            Definition 1
 
                     Term 2
                            Definition 2
 
                     Term 3
                            Definition

(Q.14) : What is the use of a span tag? Explain with example?

Ans => The <span> tag is an inline container used to mark up a part of a text, or a part of a document. The <span> tag is easily styled by CSS or manipulated with JavaScript using the class or id attribute. The <span> tag is much like the <div> element, but <div> is a block-level element and <span> is an inline element.

Example :
                          <p>My mother has <span style="color:blue">blue</span> eyes.</p>

My mother has blue eyes.

(Q.18) : What is SVG?

                     
Ans => An SVG image begins with an <svg> element
        
                     The width and height attributes of the <svg> element define the width and height of the SVG image
                     The <circle> element is used to draw a circle
                     The cx and cy attributes define the x and y coordinates of the center of the circle. If cx and cy are not set, the circle's center is set to (0, 0)
                     The r attribute defines the radius of the circle
                     The stroke and stroke-width attributes control how the outline of a shape appears. We set the outline of the circle to a 4px green "border"
                     The fill attribute refers to the color inside the circle. We set the fill color to yellow
                     The closing </svg> tag closes the SVG image
                     


                     
Example :

                                              

                     <svg width="100" height="100">
                            <circle cx="50" cy="50" r="40" stroke="green" stroke-width="4" fill="yellow" />
                     </svg>
